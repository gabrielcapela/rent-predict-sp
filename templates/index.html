Calculadora
de Aluguel

Sobre o projeto

Esta aplicação utiliza técnicas de Machine Learning para prever o valor de moradia (aluguel+condomínio) de apartamentos na cidade de São Paulo, com base em atributos como localização, metragem, número de quartos, vagas de estacionamento, entre outros fatores relevantes.

Os dados utilizados foram compilados em 2019 e tratados para o treinamento de um modelo preditivo. A aplicação está hospedada na nuvem, por meio da plataforma Heroku.

Este é um projeto pessoal desenvolvido por Gabriel Capela.
